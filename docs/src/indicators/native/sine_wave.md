# Sine Wave

Plots a sine wave and a lead-sine wave based on the cyclic phase of price movement.

## Formula


\[
\text{Sine} = \sin(\text{Phase})
\]
\[
\text{LeadSine} = \sin(\text{Phase} + 45^\circ)
\]
