# Fisher Transform

Converts inputs to a nearly Gaussian probability distribution, creating sharp peaks at turning points.

## Formula


\[
Fish(x) = 0.5 \times \ln\left(\frac{1 + x}{1 - x}\right) = \text{atanh}(x)
\]
