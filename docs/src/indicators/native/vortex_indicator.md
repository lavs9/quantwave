# Vortex Indicator

The Vortex Indicator helps identify the start of a new trend or the continuation of an existing one.

## Parameters

- `period` (default: 14): Period

## Formula


\[
VI+ = \frac{\sum VM+}{\sum TR} \\ VI- = \frac{\sum VM-}{\sum TR}
\]
