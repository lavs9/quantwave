# Bill Williams Alligator

Trend-following indicator using three delayed smoothed moving averages.

## Formula


\[
\text{Jaw} = \text{SMMA}(13, 8), \text{Teeth} = \text{SMMA}(8, 5), \text{Lips} = \text{SMMA}(5, 3)
\]
