# MAD

Moving Average Difference: 100 * (SMA(short) - SMA(long)) / SMA(long)

## Parameters

- `short_period` (default: 8): Short-term SMA period
- `long_period` (default: 23): Long-term SMA period

## Formula


\[
MAD = 100 \times \frac{SMA(short) - SMA(long)}{SMA(long)}
\]
