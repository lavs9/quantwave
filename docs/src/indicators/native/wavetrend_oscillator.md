# WaveTrend Oscillator

WaveTrend is an oscillator that helps identify overbought and oversold conditions.

## Parameters

- `n1` (default: 10): Channel Length
- `n2` (default: 21): Average Length

## Formula


\[
WT_1 = EMA(ESA, n_2)
\]
