# Indicators Guide

*   **[Ehlers DSP Suite](ehlers/index.md)**: Advanced cycle and trend tools from John Ehlers.
*   **[Advanced Indicators](advanced/index.md)**: Cutting-edge tools for market analysis.
*   **[TA-Lib Compatible](talib/index.md)**: High-performance replacements for standard TA-Lib functions.
